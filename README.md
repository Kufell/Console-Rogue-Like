# Console Rogue Like

Szkelet gry Rogue-Like/RPG z podstawami walki oraz tablicą wyników.

### Dependencies

*  Kompilator C++
*  IDE

### Uruchamianie

* Możesz otworzyć program używając swojego IDE, bądź stworzyć własny plik wykonywalny.
* Uruchamianie będzie się różnić zależnie od IDE
